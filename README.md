# Crypto Price Tracker

A simple desktop application built with **Tkinter** that allows users to check current cryptocurrency prices and percentage changes in multiple fiat currencies.

## Features

- **Multi-currency support**: USD, AUD, EUR, JPY, GBP, CNY, SGD
- **Top 20 cryptocurrencies** by market cap (via CoinMarketCap)
- **Real-time price data** (refreshed on demand)
- **Percentage change tracking**:
  - Last 24 hours
  - Last 30 days
  - Last 90 days
- Clean, easy-to-use graphical interface

## Requirements

- Python 3.8 or higher
- `requests` library
- `tkinter` (usually comes pre-installed with Python)
